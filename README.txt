Все тестовое задание выполнено с использованием таких технологий: 
- MVVM, 
- LiveData, 
- Fragment navigation (From Jetpack), 
- Coroutines, 
- Retrofit2, 
- DataBinding
- Lottie animations

Из дополнительных заданий выполнено:
- Добавлена возможность возвращаться назад по webview при нажатии кнопки назад.
- Запуск приложения в режиме full-screen, скрывая system ui.
- Добавлена поддержка переворота экрана (режимы portrait и landscape) без перезапуска webview и сохранением состояния.
- Установлена иконка приложения.

В папке apks находятся debug.apk и release.aab
